Database

เพิ่งรู้ว่าเดี๋ยวนี้มีบริการ MySQL-compatible serverless database ให้ใช้แล้ว คือพวก DBaaS หลายๆ เจ้า คิดราคาแบบเป็นรายชั่วโมงที่เปิดใช้งาน อย่างของ Google Cloud ต่อให้แทบไม่ได้ใช้งานอะไรเยอะแยะ แต่ถ้าต้องใช้บ้าง ก็ต้องจ่ายเดือนละเกือบพัน

ส่วนอันนี้ชื่อว่า PlanetScale คิดค่าใช้งานตามจำนวนข้อมูลที่เก็บและจำนวนแถวที่อ่านและเขียน… ข้อดีคือถ้าใช้งานไม่เยอะ ก็ฟรีเลย

ข้อควรระวังคือถ้ากำหนด Index ไว้ไม่ดี ก็จะเกิดการ Scan ข้อมูลหมดทั้งตาราง ถ้าเป็นฐานข้อมูลที่เราดูแลเองก็ Query เราก็จะช้าหน่อย แต่ของ PlanetScale เขาจูนระบบไว้แบบว่าลูกค้าไม่รู้สึกอะไรเลย แต่ค่าใช้จ่ายบานปลาย
