About notes.dt.in.th

Outdated

On this site, notes now have meaningful filenames; only journal entries are named with timestamp-based IDs.

I am thinking of an idea for a lightweight way to do some private journaling with rich data (custom CSS, interactivity). All data is stored as Markdown files named with meaningless ID. They are flat and there is no organization.

File name scheme is: YYMMDD "T" HHMMSS "Z" RRRR ".md". RRRR means 4 digits of random number to reduce the chance of collision and guessability.

Some extra tools will have to be created to allow easy editing and searching. I also want to eventually be able to publish some entries to the public web.
